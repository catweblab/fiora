A) Installation - top

Before you install Fashionista Street Template, make sure you have installed and running the correct version of OpenCart. IF YOU ARE UPGRADING BACKUP YOUR SYSTEM BEFORE UPLOADING FILES.
There are three folders to be uploaded in the installation directory that you will found in upload directory

  • Admin
  • Catalog
  • image

While uploading you will be asked to overwrite some of the files from your current installation. Overwrite all files.

After the  files are uploaded. Go to the administration panel > system > settings >  choose the fashionista template from the Store tab.

B) Module Installation - top

3 module have been developed for this theme. The one is for customizing theme color variations, fonts, payment image, backgrounds, social links etc. An the other two module are developed for highlighting featured contents: Featured Banners and Featured Content Banners that can be shown in any layout

  1. Fashionista Street Theme Options
  2. Fashionista Banners
  3. Fashionista Content Banners

B - 1) Fashionista Street Theme Options - Module Installation

When the template is activated to use the theme option (to manage social links, icons, font settings, backrounds, payment images)  go to extensions > modules > Fashionista Street Theme Options and click on install. After the theme option module is installed go to edit and customize the theme as you like and click on save and activate.

B - 2) Fashionista Banners - Module Installation

When the template is activated to manage banners  go to extensions > modules > Fashionista Banners and click on install. After the Banner module is installed go to edit and customize the banners as you like and click on save.

Banner Image Resolution : 280x150

B - 3) Fashionista Content Banners - Module Installation

When the template is activated to manage content banners  go to extensions > modules > Fashionista Content Banners and click on install. After the Content Banner module is installed go to edit and customize the banners as you like and click on save.

Image Resolution : 90x90

C) Image Settings - top

When the template is activated, go to System > Settings > Image tab and adjust the image settings with the following values:

Slideshow Dimension Settings

  • 1 Column > 900x400
  • 2 Column > 717x400

Category Image Dimension Settings

  • 1 Column > 900x200
  • 2 Column > 717x200

Modules Image Dimension Settings

  • Carousel : 120x180
  • Bestsellers : 160x239
  • Featured : 160x239
  • Latest : 160x239
  • Specials : 160x239
  • Fashionista Banners : 280x150
  • Fashionista Content Banners: 90x90

To Display titles on slideshow, go to system > design > banners > edit (the banner that you are using for slideshow) and modify or add the titles.

D) CSS - top

In case you want to modify CSS to customize more, you will find the CSS files in stylesheet folder

  1. stylesheet.css (Default Skin - Pink)
  2. carousel.css
  3. slideshow.css
  4. ie6.css
  5. ie7.css

E) Javascript - top

The theme has 2 additional javascript files

  1. custom-scripts.js
  2. product-zoom.js (For Zoom Effect)

custom_scripts.js is used for controlling different visuals and functional modifications

  • Tooltip Hover Effect
  • Scroll top Top
  • Image Hover Fade Effect
  • Loading Background Effect
  • Product List Rollover Effect
  • Twitter Ticker
